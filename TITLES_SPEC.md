# 칭호(타이틀) 명세서

`app/lib/titles.ts` 의 규칙과 1:1 대응. **숫자만 고쳐서 돌려주시면** 코드에 반영합니다.
구조/이름/아이콘 바꾸고 싶은 것도 메모로 적어주세요.

- **등급형**: 누적값이 쌓이며 등급 상승. `[루키, 아마추어, 준프로, 프로]` 순 임계값.
- **달성형**: 조건 충족 시 획득(등급 없음).
- 상태: 🟢 `live`(지금 자동) · 🟡 `future`(앞으로 치를 경기부터) · 🟣 수동(titles 시트, 코드 아님)
- ✦ = 커스텀 SVG 대표 후보

> 라인업/로스터엔 **최고등급 상위 3개**만 아이콘으로 노출, 개인 페이지엔 전부 카드로.

## 12. 수동 칭호 (🟣 titles 시트 — 코드 아님)
관리자가 Google Sheet `titles` 탭(`A=이름, B=칭호, C=lucide아이콘, D=설명`)에 직접 입력.
예: 분위기 메이커 / 패션왕 / 지각대장 / 회식 부장 / 멘탈갑 / 라커룸 리더 / 팀닥터 / 포토그래퍼 / 살림꾼 / 비주얼 담당 …

> 이 시트 연동(4단계)은 위 자동 칭호 확정 후 진행.
